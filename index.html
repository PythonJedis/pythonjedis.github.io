What/Who are the Python Jedis?

Written by Valeness on April 19, 2015

The Python Jedis began as a python study group thought up by _thelichking_ on Reddit! At first, it was a few people on skype chatting about how to get started programming, showing off projects, and just general chat about python. Eventually the idea came up to start a github repo/organization so that the members could all contribute to simple projects to help themselves learn. This great idea didn't take too long to start off, and now we have PythonJedis@Github to show for it.

Currently the github organization has around 10 members and 2 admins. Myself and Simon included. The skype group currently has 31 members and is growing every week.

To join all you have to do is message myself or any of the other github members, or grab me on skype. My username is valeness84. Furthermore, to join the github organization, all you need to do is somehow get me your github username. Either email me, github message me, skype me, or send me a carrier pigeon.

More information about how to join, how this site was made, our current projects, and member information can be found in the readme of this repository, or on the other pages of the site. (Yet to come!)
